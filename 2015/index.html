Proyecto Poncho

Base de html y css basada en Bootstrap para la creación de sitios pertenecientes a la Administración Pública Nacional de la República Argentina.

El código y la documentación de la v0.3 de Poncho están en desarrollo. Se aceptan issues con dudas o sugerencias. Para aportar con código, por favor generar un issue antes del pull request.
