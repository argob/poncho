Proyecto Mencho

Base de html y css basada en Bootstrap para la creación de sitios pertenecientes a la Administración Pública Nacional de la República Argentina.

Dependencias

El archivo mencho.css no tiene dependencias directas fuentes ni imágenes.

Para que se vea y funcione bien deben estar :

  • bootstrap.css
  • roboto-fontface.css
  • droid-serif.css
  • jquery.js
  • bootstrap.js
