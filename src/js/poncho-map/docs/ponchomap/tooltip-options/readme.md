# Tooltip options

Opciones por defecto para la presentación y posición de la etiqueta emergente. 

Para más información sobre las opción del tooltip. Visite: [Leaflet Documentation - Tooltip Options](https://leafletjs.com/reference.html#tooltip)

```js
tooltip_options:{
    className: "leaflet-tooltip-own",
    direction: "auto",
    offset: [13,-18],
    opacity: 1,
    permanent: false,
    sticky: false
}
```
