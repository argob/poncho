## Valores por defecto
```js
‘marker_cluster_options’: {  
    ‘spiderfyOnMaxZoom’: true,  
    ‘showCoverageOnHover’: false,  
    ‘zoomToBoundsOnClick’: true,  
    ‘maxClusterRadius’: 10,  
    ‘spiderfyDistanceMultiplier’: 1.5,  
    ‘spiderLegPolylineOptions’: {  
        ‘weight’: 1,  
        ‘color’: “#666”,  
        ‘opacity’: 0.5,  
    }  
}
```
