# Métodos

| Método | Retorno | Descripción |
|:--|:--|:--|
| filteredEntries | `object` | Contiene las entradas después de aplicar los filtros activos. |
| totals | `object` | Retorna la cantidad de marcadores agrupados por cada filtro, incluyendo su posición en la colección. |
| formFilters | `object` | Retorna los filtros actualmente seleccionados, indicando el grupo de filtro y su índice dentro del grupo. |
