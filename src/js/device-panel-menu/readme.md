# Device panel menu

> [!NOTE]
> Este script no está incluido en poncho.min.js. Su uso es exclusivo de algunos nodos del CMS Drupal.



En la página de libro del CMS Drupal para www.argentina.gob.ar, la función panelMenu() se ejecuta durante los eventos onLoad u onResize de la ventana para determinar la visualización del menú de navegación secundaria. En entornos de escritorio (desktop), el menú se presenta expandido en el margen izquierdo de la pantalla; por el contrario, en dispositivos móviles, el componente se muestra colapsado, permitiendo al usuario desplegarlo para interactuar con sus diferentes ítems.
