Detalle de elementos

Encabezado

Usar este encabezado para agrupar los datos y separarlos de otros para una lectura más cómoda.

Correo electrónico

Sirve para contactarse por teléfono con el usuario.

En general es mejor pedir el teléfono movil del ciudadano ya que es más personal.

Funcionamiento

Este dato está compuesto por un campo para el código de área de la localidad (incluye el número "0") y un campo para el número de teléfono (incluye el número "15").

Valida formato de mail. Hay que desactivar la validación de html5 del input email. No pedir mail 2 veces (se debe poder ingresar al sistema sin validar mail, y ahí dentro poder cambiarlo si lo puso mal).

Teléfono móvil

Sirve para contactarse por teléfono con el usuario.

En general es mejor pedir el teléfono movil del ciudadano ya que es más personal.

Funcionamiento

Este dato está compuesto por un campo para el código de área de la localidad (incluye el número "0") y un campo para el número de teléfono.

Acepta puntos, guiónes, signo mas y barras, pero envía sólo números. Valida cantidad mínima de caracteres para mostrar error donde se le pregunta al usuario si puso su código de área.

Teléfono fijo

Sirve para contactarse por teléfono con el usuario.

Pedir teléfono fijo en vez de móvil en el caso de que por ejemplo se necesite una confirmación telefónica sobre un domicilio.

Funcionamiento

Acepta puntos, guiónes, signo mas y barras, pero envía sólo números. Valida cantidad mínima de caracteres para mostrar error donde se le pregunta al usuario si puso su código de área.
