Login y registro

Datos para que el usuario pueda acceder a una plataforma con información o acciones personalizadas.

Mi Argentina

Para registrar usuarios en una plataforma, la misma debe integrar el sistema de login y registro de Mi Argentina.

Mi Argentina ofrece al usuario la posibilidad de acceder a diferentes servicios del Estado mediante un único usuario y contraseña. Y ofrece a los organismos una herramienta ya desarrollada que ahorra tiempo de implementación.

Para más información de cómo integrar una plataforma nueva o existente consultá la documentación de Mi Argentina.

Alternativa de login y registro

Si en una plataforma existente por alguna razón se debe mantener el sistema propio de login y registro, se deben pedir los datos necesarios siguiendo los lineamientos de las otras páginas de patrones, y los lineamientos de contraseñas que usa Mi Argentina.

Detalle de elementos

Contraseña (para registro)

Sirve para cuando el usuario tiene que generar una nueva contraseña.

Funcionamiento

Siempre deben mostrarse las reglas de contraseña antes de que el usuario pueda escribirla.

No pedir al usuario que escriba la contraseña 2 veces (la plataforma debe tener una forma de recuperar la contraseña a través de correo electrónico).

Contraseña (para ingresar)

Sirve para cuando el usuario tiene que ingresar a la plataforma con un usuario ya registrado.

La diferencia con el campo de contraseña para registrar, es que el de ingresar no valida si el usuario escribió algo que no siga las reglas de contraseña.

Funcionamiento

Siempre deben mostrarse las reglas de contraseña antes de que el usuario pueda escribirla.

No pedir al usuario que escriba la contraseña 2 veces (la plataforma debe tener una forma de recuperar la contraseña a través de correo electrónico).
